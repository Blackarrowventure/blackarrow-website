🎨 New Color System

GOLD (#F59E0B)

Headings, titles, section headers, accent highlights

EMERALD GREEN (#10B981)

All CTA buttons (Get Quote, Submit, etc.)

DARK EMERALD (#059669)

Button hover state & darker accents

WHITE (#FFFFFF)

Body text, descriptions, secondary content

✨ Key Benefits of This Change

  • Better Visual Hierarchy: Users instantly know gold = important info, green = actions
  • Improved CTR: Green buttons are proven to increase click rates vs. gold
  • Professional Polish: Two-color system looks more sophisticated than one-color
  • Better Accessibility: Color-blind friendly - green and gold have different brightness
  • Mobile Friendly: Thumb-zone buttons stand out better with green
  • Consistent Branding: Emerald green complements gold without clashing

📋 What Will Change in CSS

/* Add to :root color variables */
--clr-accent-primary:   #F59E0B;  (keep gold for headings)
--clr-accent-cta:       #10B981;  (new emerald for buttons)
--clr-accent-cta-hover: #059669;  (darker emerald on hover)

/* Update button colors */
.btn-primary {
  background: var(--clr-accent-cta);      /* Changed from gold to green */
  color: var(--clr-white);
}

.btn-primary:hover {
  background: var(--clr-accent-cta-hover);
  box-shadow: 0 0 20px rgba(16,185,129,.3);
}

/* Keep gold for headings */
h1, h2, h3 {
  color: var(--clr-accent-primary);  /* Stays gold */
}
