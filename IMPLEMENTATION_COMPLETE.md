**HTML Structure Updated:**

All images now use `<picture>` elements with WebP + JPG fallback:

```html
<picture>
  <source srcset="assets/images/services/service.webp" type="image/webp">
  <img src="assets/images/services/service.jpg" alt="...">
</picture>
```

**How it works:**
- ✅ Modern browsers (Chrome, Firefox, Edge, Safari 16+) load WebP (9.6% smaller)
- ✅ Older browsers fall back to JPG automatically
- ✅ No user impact - seamless transition
- ✅ Better loading performance on mobile
